# vereins-buchhaltung
Eine doppelte Buchhaltung mit Kontenrahmen für Schweizer Vereine. Konten können einfach angepasst werden, Buchhaltungsassistent mit Claude für Belegsverarbeitung. Pull-Requests willkommen. Super schnell, läuft alles im Browser und speichert in local storage. Backup und Import mit JSON. Einfachstes Deployment: die HTML-Datei im Browser öffnen. 

Auf der Website https://claude-buha.werckmeister.app/Werckmeister_Buchhaltung.html läuft eine Instanz. 

Für die Claude-Belegsverarbeitung im das config.example.json duplizieren nach config.json und den Claude-API-Key eintragen. Ist im .gitignore drin.
